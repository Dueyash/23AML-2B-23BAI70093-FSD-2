# Experiment 5.1: Implement Lazy Loading in SPA

## Aim

To design a React application using **Lazyloading** , **Suspense** and **React Router DOM** components.

## Tools & Libraries

- React (Vite)
- Lazy loading
- Suspense
- React Router DOM

## Description

This project demonstrates a single React application that includes:

- A **Dashboard**
- **Components to switch**
